# RollDice

## Usage
`virtualenv venv`<br/>
<br/>
Windows             
`venv\Scripts\activate`<br/>
Linux              
`source venv\bin\activate`<br/><br/>

`pip install -r requirements.txt`<br/><br/>

`python manage.py makemigrations`<br/>
`python manage.py migrate`<br/><br/>

`python manage.py runserver`<br/><br/>

Enjoy
